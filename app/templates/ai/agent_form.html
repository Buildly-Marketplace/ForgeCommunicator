Where to get your API key

OpenAI (ChatGPT) - Best for general tasks, code, and creative writing

  1. Go to platform.openai.com/api-keys
  2. Sign in or create an account
  3. Click "Create new secret key"
  4. Copy the key (starts with sk-)

Pricing: Pay-as-you-go. GPT-4o is ~$5/1M input tokens. View pricing

Anthropic (Claude) - Best for analysis, research, and nuanced responses

  1. Go to console.anthropic.com/settings/keys
  2. Sign in or create an account
  3. Click "Create Key"
  4. Copy the key (starts with sk-ant-)

Pricing: Pay-as-you-go. Claude 3.5 Sonnet is ~$3/1M input tokens. View pricing

Perplexity AI - Best for research with real-time web search

  1. Go to perplexity.ai/settings/api
  2. Sign in (requires Perplexity Pro subscription)
  3. Click "Generate" under API Keys
  4. Copy the key (starts with pplx-)

Pricing: Requires Pro ($20/mo) + API credits. ~$0.20/1M input tokens. View pricing
